Una tarea está "terminada" cuando:

- Funciona correctamente.
- Fue probada y no presenta errores.
- Quedó registrada en Git con su respectivo commit.

Evento que activa el flujo:
Nuevo reporte registrado en el sistema.

Mensaje que se enviaría a Telegram:
"Nuevo reporte recibido. Ingrese al sistema para revisarlo."

Nombres completos, documentos de identidad, direcciones, teléfonos, contraseñas o cualquier dato personal sensible.
